# Plan_event. Telegram-бот для автоматизации проектирования мероприятий

MVP-прототип интеллектуального помощника для организаторов событий, вожатых, методистов и ведущих. Бот пошагово собирает ключевые параметры будущего мероприятия и мгновенно формирует лаконичный план с креативными фишками и точным таймлайном.

## Продуктовая ценность
- **Для организатора:** Избавляет от проблемы «чистого листа», экономя часы рутинной работы при составлении поминутных планов.
- **Масштабирование опыта:** Проект выступает цифровым мостом, упаковывая практический вожатский опыт автора и академическую базу научно-исследовательских работ (НИР) по управлению вниманием аудитории в готовый прикладной ИИ-инструмент.

## Технологический стек
- **Язык разработки:** Python 3.10+
- **ИИ-ядро:** Сбер GigaChat API
- **Оркестрация ИИ:** LangChain (`langchain_gigachat`, `langchain_core`)
- **Интерфейс:** pyTelegramBotAPI (`telebot`)

## Ключевые особенности архитектуры
- **Пошаговый UX (FSM-диалог):** Реализован линейный сбор данных от пользователя (Название -> Описание задачи -> Длительность) с удержанием контекста внутри текущей сессии.
- **Оптимизированный Prompt Engineering:** Промпты для модели GigaChat настроены на жесткое соблюдение формата. ИИ возвращает строго структурированный, лаконичный ответ без канцеляризмов, псевдонаучного шума и лишнего текста.

## Инструкция по запуску
1. Склонируйте репозиторий и установите необходимые библиотеки:
   ```bash
   pip install pyTelegramBotAPI langchain-gigachat langchain-core
   ```
2. Укажите ваши актуальные ключи авторизации в коде:
   - `TELEGRAM_TOKEN` — токен вашего бота из @BotFather
   - `GIGACHAT_CREDENTIALS` — API-ключ из личного кабинета Сбер Студии
3. Запустите скрипт
